# DSSim Documentation

DSSim is a discrete-event simulation framework for building process, queue, and resource models.

## What You Get Here

- Practical guides for `LiteLayer2` and `PubSubLayer2`
- Concept docs for architecture and scheduling
- Technical user guide chapters with glossary
- API reference pages generated from code
- Pointers to runnable examples

## Quick Start

1. Install project dependencies.
2. Run examples in `examples/lite` or `examples/pubsub`.
3. Use the guides in this documentation to map your model to the right layer.
4. Read the [Technical User Guide](user-guide/index.md).

## Media Support

This documentation stack supports:

- Images (`png`, `jpg`, `svg`)
- Animated media (`gif`, `webm`, `mp4`)
- Mermaid diagrams for sequence and flow visualization

Use `docs/assets/images`, `docs/assets/animations`, and `docs/assets/diagrams` for assets.
