# Salabim Parity

Parity support includes a Salabim adapter plus project-level model variants for side-by-side comparison.

See:

- adapters under `dssim/pubsub/parity/`
- Salabim model variants under `projects/` (for example `crossroad_salabim.py`, `cpu_cache_salabim.py`)

This helps confirm conceptual equivalence across frameworks, even when the example form differs from SimPy parity scripts.
